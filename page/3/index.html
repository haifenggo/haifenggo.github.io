矩阵加速

让我们观察一下这题的递推式:

f[1] = f[2] = f[3] = 1f[1]=f[2]=f[3]=1

f[n] = f[n - 1] + f[n - 3]f[n]=f[n−1]+f[n−3] (n > 3n>3)

首先我们可以发现，f[n]f[n] 最早由f[n-3]f[n−3]转移而来，所以我们需要构造一个3*33∗3的矩阵。

首先，我们将与 f[n]f[n] 有关的需要得到的式子放入一个矩阵: \begin{bmatrix}f[n]\f[n - 1]\f[n - 2]\end{bmatrix}f*[n]f[n−1]f[n−2]

然后，我们寻找他从哪里来: f[n-1] = f[n - 2] + f[n - 3]f[n−1]=f[n−2]+f[n−3]， 即 \begin{bmatrix}f[n-1]\f[n - 2]\f[n - 3]\end{bmatrix}f*[n−1]f[n−2]f[n−3]

然后设我们需要得到的矩阵为ansans，有:ans\begin{bmatrix}f[n-1]\f[n - 2]\f[n - 3]\end{bmatrix} = \begin{bmatrix}f[n]\f[n - 1]\f[n - 2]\end{bmatrix}ans∗f[n−1]f[n−2]f[n−3]=f*[n]f[n−1]f[n−2]

拓扑排序

拓扑排序步骤如下：
（1）定义一个队列Q，并把所有入度为0的结点加入队列
（2）取队首结点，访问输出，然后删除所有从它出发的边，并令这些边到达的顶点的入度减1，如果某个顶点的入度减为0，则将其加入队列。
（3）重复进行（2）操作，直到队列为空。如果队列为空时入过队的结点数恰好为N，说明拓扑排序成功，图G为有向无环图；否则，拓扑排序失败，图G有环。

欧拉路径

  • 首先我们要确定出起点，然后从起点开始不走重复边的递归。

  • 对于每一个点,当访问完所有的边时，把这个点加入答案序列。

  • 最后，输出答案序列即为一条欧拉路径依次经过的点。

Bellman-Ford算法

目的: 解决含负权边的单源最短路径问题，同时可以判断是否存在负权回路

（1）初始化：将除起点s外所有顶点的距离数组置无穷大 d[v] = INF, d[s] = 0
（2）迭代：遍历图中的每条边，对边的两个顶点分别进行一次松弛操作，直到没有点能被再次松弛
（3）判断负圈：如果迭代超过V-1次，则存在负圈

SPFA（Bellman-Ford算法改进版本）

思想： 松弛操作必定只会发生在最短路径前导节点松弛成功过的节点上，用一个队列记录松弛过的节点，可以避免了冗余计算。
